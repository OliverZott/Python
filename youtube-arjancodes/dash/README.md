# Dash tutorial

<https://www.youtube.com/watch?v=XOFrvzWFM7Y&t=217s>

## Prerequisites

create venv

```bash
python -m venv venv
```

activate env

```bash
.\venv\Scripts\activate
```

deactivate env

```bash
.\venv\Scripts\deactivate
```

install packages and check

````bash
pip install --upgrade -r requirements.txt
pip list
````

## Run

```shell
python .\main.py
```
